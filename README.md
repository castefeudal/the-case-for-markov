# MARKOVMADE — GitHub Pages fixed build

## Публикация
1. Создайте репозиторий.
2. Загрузите **всё содержимое этой папки** в корень репозитория, не саму внешнюю папку.
3. Проверьте, что `index.html` лежит в корне.
4. GitHub → Settings → Pages → Deploy from a branch → `main` / `(root)`.
5. Подождите 1–3 минуты и откройте выданный GitHub Pages URL.

## Что исправлено
- сайт запускается без Tailwind CDN;
- основной контент никогда не остаётся скрытым загрузчиком;
- изображения имеют локальные резервные версии;
- внешние изображения подгружаются только после успешной проверки;
- мобильное меню не блокирует страницу;
- курсор остаётся доступным при сбое JavaScript;
- все кнопки получили корректный `type`;
- ссылки в новой вкладке защищены `noopener noreferrer`;
- добавлены `.nojekyll` и `404.html`.

## Важно
Не переименовывайте `index.html`. Не загружайте на GitHub только TXT-файл: GitHub Pages открывает сайт именно из `index.html`.
