# PAVEL MARKOV — Portfolio / Offer Edition

Финальная статическая сборка персонального сайта под роли **Project Manager / Operations Manager**.

## Состав

- `index.html` — готовый сайт;
- `Pavel_Markov_Portfolio_Code.txt` — тот же исходный код в TXT;
- `hero-head2.mp4` — оптимизированное интерактивное видео;
- `hero-head2-poster.jpg` — постер видео и резервное отображение;
- `pavel-profile.jpg` — локальное изображение профиля;
- `pavel-motion.jpg` — локальное изображение для проектного раздела;
- `.nojekyll` — отключает обработку файлов Jekyll на GitHub Pages.

## Установка в GitHub-репозиторий

1. Скопировать все файлы из архива в корень репозитория.
2. Старый `index.html` заменить новым.
3. Убедиться, что имя видео строго `hero-head2.mp4`.
4. В GitHub открыть **Settings → Pages**.
5. Выбрать **Deploy from a branch → main → /(root)** и сохранить.
6. После публикации выполнить жёсткое обновление страницы: `Ctrl + F5`.

## Как работает интерактивный портрет

На компьютере положение курсора выбирает естественные фрагменты движения головы: влево, вправо, вверх, вниз и центр. Переходы выполняются через временную шкалу видео без Canvas, нейросетей и тяжёлых библиотек. На сенсорных устройствах используется спокойное циклическое воспроизведение. При включённом системном режиме уменьшения движения остаётся статичный постер.

Видимая область видео заранее кадрирована: отметка Kling AI не входит в кадр. Сам файл оптимизирован для веб-загрузки и содержит частые ключевые кадры для плавного управления курсором.
