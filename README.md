# Pavel Markov Portfolio — FULL WORKING RELEASE

## Запуск

1. Не меняйте структуру папки.
2. Откройте `index.html` в браузере для локальной проверки.
3. Для GitHub Pages загрузите **все файлы из этой папки в корень репозитория**.
4. В GitHub откройте `Settings → Pages`, выберите `Deploy from a branch`, ветку `main`, папку `/root`.

## Обязательные файлы

- `index.html` — полный сайт;
- `head-idle.mp4` — спокойная анимация портрета;
- `head-left.mp4`, `head-right.mp4`, `head-down.mp4` — управляемые состояния головы;
- `hero-head-cinematic-poster.jpg`, `hero-head2-poster.jpg` — постеры и резервные изображения;
- `pavel-profile.jpg`, `pavel-motion.jpg` — изображения разделов;
- `.nojekyll` — корректная публикация через GitHub Pages.

## Проверенный функционал

- адаптивный первый экран;
- интерактивный портрет и плавное смешивание видеослоёв;
- RU / EN;
- HR-режим «15 секунд»;
- калькулятор бизнес-эффекта;
- мобильное меню;
- печать / PDF;
- scroll progress, reveal-анимации и навигация;
- отсутствие горизонтального переполнения на 1440 px и 390 px.

## Важно

Один только `index.html` не содержит сами видео. Для полной работы необходимо загрузить весь комплект файлов из архива.
